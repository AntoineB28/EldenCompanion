# Elden Ring D&D Companion ⚔️

An Android companion application custom-built for **Antoine Boisclair's Elden Ring-inspired D&D one-shot campaign series**. This tool bridges the gap between tabletop mechanics and video game tracking to keep combat fast-paced and immersive.

## ✨ Key Features
* **Player Dashboard:** Real-time tracking for Player Life (HP) and custom Mana systems.
* **DM Screen:** A dedicated interface for the Dungeon Master to manage epic boss health bars and encounter status.

## 🚀 Download & Installation
Get ready for the boss fight by downloading the latest version:
1. Head over to the **[Releases](../../releases)** section.
2. Download the latest `elden-companion.apk` or `elden-companion.msi`.
3. Open the file on your Android device to install it (ensure "Install from Unknown Sources" is enabled).
